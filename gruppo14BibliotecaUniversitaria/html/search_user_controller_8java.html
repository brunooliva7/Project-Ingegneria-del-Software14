Detailed Description

Controller per la ricerca di un utente nel sistema.

La classe searchUserController gestisce l'interfaccia utente per la ricerca di un utente tramite la sua matricola, nome o cognome. Consente di visualizzare le informazioni dell'utente, tra cui prestiti attivi, all'interno di una TableView. La ricerca avviene sulla base dell'input fornito e i risultati vengono mostrati in tempo reale.
