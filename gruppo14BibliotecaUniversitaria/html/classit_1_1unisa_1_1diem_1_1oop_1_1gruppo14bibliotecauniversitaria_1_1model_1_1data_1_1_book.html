Detailed Description

Gestisce i dati dei libri della biblioteca.

La classe Book gestisce i dati principali di un libro: titolo, autori, anno di pubblicazione, codice ISBN e numero di copie disponibili Implementa l'interfaccia Comparable per consentire l'ordinamento per titolo

Invariant
availableCopies >= 0
See also
Comparable

◆ compareTo()

int it.unisa.diem.oop.gruppo14bibliotecauniversitaria.model.data.Book.compareTo ( Book other)

Confronta due libri per ordinamento alfabetico dei titoli.

L'ordinamento avviene in base al titolo (case-insensitive) Se i titoli sono uguali, il confronto viene effettuato sugli autori (case-insensitive)

Parameters
otherAltro libro da confrontare con quello corrente
Returns
Un valore intero: negativo se il titolo di other precede quello corrente; zero, se sono uguali; positivo, se il titolo di other segue quello dell'oggetto corrente
Precondition
other != null (se other è null viene sollevata NullPointerException)
Postcondition
Restituisce un valore coerente con l'ordinamento alfabetico dei titoli e degli autori
