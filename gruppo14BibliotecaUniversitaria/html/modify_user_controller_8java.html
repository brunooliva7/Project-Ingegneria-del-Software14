Detailed Description

Controller per la gestione della modifica di un utente esistente.

La classe modifyUserController gestisce l'interfaccia utente per la ricerca di un utente e la modifica delle sue informazioni. Consente di cercare un utente, visualizzare le sue informazioni, e di modificarle. I dati vengono aggiornati nel modello e la TableView viene aggiornata automaticamente.
