Detailed Description

Rappresenta un prestito di un libro da parte di un utente.

La classe Loan associa un libro ad un utente e ne registra la data di restituzione. Implementa l'interfaccia Comparable per consentire l'ordinamento dei prestiti secondo un criterio definito (ad esempio per data di scadenza).
